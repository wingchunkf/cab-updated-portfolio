Experience

Junior Front-End Developer

American Solutions, LLC
June 2017 through April 2018

Working under a mentor, I was involved in two major projects. The first was a template-based site for a local home renovation/construction company, Castle Rock Construction. The site can be viewed here. For this project, I reorganized the layout of each page to suit the needs of the client, chose a color palette and created the corresponding Sass variables, added images, implemented Google Analytics, added a Favicon, and created a build pipeline for compiling Sass files.

For the second project, which was much larger in scope, I was responsible for a complete revamping of the website accompanying a security company management app built and owned by my mentor. At that time, the app was being offered free of charge and was utilized by approximately 500 users per day. The goal of this project was to revamp the website and app for monetizing purposes. Working from basic Justinmind prototypes, I coded the HTML, styled the six-page site, and used JavaScript, Vue, and jQuery to add functionality and interactivity.

My mentor also tasked me with cross-browser compatibility testing, testing the site on various devices and operating systems, and finding and eliminating bugs throughout the entire process. In addition, I implemented Google Analytics for the project and completed many SEO-related tasks. Multiple tools and frameworks were used for this effort, including Bootstrap 4, Sass, Gulp, Vue JS, Photoshop, Git, Visual Studio Code, the Git Bash CLI, JavaScript, and jQuery. The project was managed in Jira, stored in a Bitbucket repository, and developed in two-week sprints using an Agile approach. Over the course of both projects, I gained a plethora of technical skills and also experienced the software development lifecycle as a member of a development team.

 Working with Christine was both a valuable and rewarding experience. Christine is an articulate and talented web developer. I've had the pleasure to work with her on multiple projects and truly appreciate her ability to move through tasks quickly as well as adding her touch of creativity. I would highly recommend Christine to anyone in the market for web development or web services! 

Cory Robinson, Fullstack Developer/Former Mentor

Work

Star Wars Trivia App

This app utilizes SWAPI, an API built by Paul Hallett and team. Users may choose from six different categories, Characters, Species, Planets, Starships, Vehicles, and Films. Asynchronous data requests are made with the HTTP client Axios. Tools used include JavaScript, jQuery, Axios, Gulp, Sass, and Bootstrap 4. The project was planned and tracked in Jira using an Agile approach. This is a completely original project, no tutorials or Free Code Camp.
