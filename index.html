My Work

Star Wars Trivia App

This is a Star Wars trivia app that utilizes the Star Wars API (SWAPI) built by Paul Hallett and team. Users may choose from six different categories, Characters, Species, Planets, Starships, Vehicles, and Films. Asynchronous data requests are made with the HTTP client Axios. Tools used include JavaScript, jQuery, Axios, Gulp, Sass, and Bootstrap 4. The project was planned and tracked in Jira using an Agile approach. This is a completely original project, no tutorials or Free Code Camp.
